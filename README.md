# Research

原语里弄课题研究聚焦基础性、前瞻性问题，旨在探索技术与应用及行业生态的相互作用，并提升公众对前沿技术的理解。课题倾向于进行开创性探索和实践，并以多种形式输出成果，包括而不限于研究报告、播客、互动式游戏。
